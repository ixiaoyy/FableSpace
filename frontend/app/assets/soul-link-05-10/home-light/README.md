# Home light runtime materials

This directory is the light-theme counterpart of `../home-black/` for the SoulLink home/reference artboard.

## Rule

- Home page runtime materials live under `home-light/` or `home-black/` by theme.
- Keep file names slot-based (`hero-*`, `node-*`, `guide-*`, `icon-*`, `bg-*`, `invite-card*`) so light/black imports can follow the same material-slot rule.
- Do not import runtime homepage materials from ad-hoc scratch folders such as `user-cuts-light/`.
- User-provided original reference cuts remain provenance; accepted runtime copies belong in this page/theme directory and are tracked in `../manifest.json`.
