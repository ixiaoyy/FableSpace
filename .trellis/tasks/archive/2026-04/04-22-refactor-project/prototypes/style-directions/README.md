# FableMap 多风格视觉方向板

这些方向板用于说明：P1.4 前端拆分时，页面信息架构是稳定的，但酒馆视觉可以主题化。每个主题都必须服务同一条产品主链路：真实坐标 → 店主酒馆 → AI NPC → 记忆回访。

## 方向

1. [暖木质赛博酒馆](./01-warm-cyber-tavern.svg)
   - 温暖、手工、铜边、羊皮纸、烛火。
   - 适合默认主题或新手友好酒馆。

2. [霓虹夜城赛博朋克](./02-neon-megacity-cyberpunk.svg)
   - 黑黄高对比、雨夜、故障字、霓虹招牌、密集街区。
   - 可参考“硬核赛博都市”的情绪，但不能复刻 Cyberpunk 2077 的具体 UI、Logo、字体、地图、角色或商标元素。

3. [小清新日式恋爱漫画](./03-fresh-japanese-romance.svg)
   - 粉/薄荷/浅蓝、柔和纸感、恋爱便签、街角咖啡/酒馆。
   - 适合治愈、轻社交、校园附近或约会感酒馆。

4. [手绘幻想小镇动画感](./04-handpainted-fantasy-town.svg)
   - 水彩天空、绿意、小镇路牌、童话感回访记忆。
   - 可吸收“温柔手绘动画”的宽泛情绪，但不要模仿任何具体在世艺术家的个人笔触、角色比例或专有场景。

## 设计规则

- 多风格是 theme skin，不改变 API、数据模型、主链路和权限边界。
- 每个主题都必须保留真实地图锚点、店主主权、AI NPC 对话、记忆反馈、owner secrets 不外泄。
- 后续可把主题沉淀为 design tokens：色板、材质、按钮、卡片、地图标记、对话气泡、owner console 控件。
