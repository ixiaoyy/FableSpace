# Special Tavern Type Roadmap

## Goal

Record a lightweight roadmap for future special tavern types that can reuse the thin type layer and play-pack model after cultivation taverns prove the pattern.

## Todo

* List candidate types: divination, detective, short-drama, urban-legend, public-welfare companion, relationship-graph tavern.
* For each type, define the compatible play-pack shape and forbidden scope.
* Keep roadmap non-blocking; do not require framework work before the cultivation MVP.

## Acceptance Criteria

* [ ] Roadmap clarifies reuse without expanding MVP scope.
* [ ] Each candidate respects FableMap constraints: real coordinates, owner confirmation, no generic social/game economy.
* [ ] The roadmap can be converted into separate Trellis tasks later.

## Out of Scope

* Implementing additional types now.
* Plugin marketplace or user-installed arbitrary scripts.
