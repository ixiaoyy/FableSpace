# AIO3 世界记忆图谱认领

## 认领时间
2026-03-17

## 任务描述
设计玩家、POI、zone、route、echo、历史片段与阵营关系的统一记忆索引结构。

## 现有基础
- `fablemap/memory_graph.py` - 已有 Player-POI 关系基础实现

## 扩展范围
1. Zone 记忆（区域熟悉度、停留时长）
2. Route 记忆（路径追踪、重复路径识别）
3. Echo 记忆（回声关联、历史片段）
4. Faction 关系（阵营倾向、声望）
5. 统一查询接口

## 最小化实现
- 扩展现有 dataclass 结构
- 不引入数据库
- 保持内存存储
