# FableSpace 明确不做清单

本文档是方向约束。新需求如果碰到这些边界，默认不做；确需突破时必须先更新产品文档和 Schema，再实现。

## 核心判断

FableSpace 的价值来自：

- 故事正史与连续性 > 运行时临场生成
- 角色差异 > 统一话术模板
- 真实锚点 > 无限自由生成
- 空间 / NPC 关系 > 通用社交网络
- SillyTavern 兼容和可导出 > 平台锁定

## 不做 1：运行时 AI 擅自改写已发布正史

不做：

- AI 在对话中把访客谎言、跨世界能力或临时发挥写成 Space 正史。
- 未经评审覆盖、改写或扩写已发布的 NPC、世界书、事件或玩法。
- 用统一模板替代角色自己的欲望、秘密、语言和代价。

允许：

- 经过人工评审、版本化并可回滚的系统故事种子。
- 导入和导出 SillyTavern 角色卡。
- 生成未发布、可编辑、可丢弃的候选草稿。

硬规则：候选草稿在明确发布前不得进入公开 Space payload，不得覆盖已有 NPC，也不得随空间包导出。

## 不做 2：平台级 Token 付费系统

不做：

- 平台内充值、购买、结算、抽成。
- 平台级 Token 市场。

既有自托管 LLM / API Key 和 owner 可见统计可作为兼容能力保留，但不是当前产品主线。

## 不做 3：无真实坐标的自由空间

不做：

- 脱离真实地图的自由幻想空间。
- AI 创造坐标或漂浮世界。

允许：

- 空间挂接真实 `lat/lon` 后进行主题化、氛围化表达。
- 公开展示时按隐私需求做模糊或抽象表达，但底层仍必须有真实锚点。
- 首页按角色优先展示，但角色必须明确归属于一个保留真实坐标、访问规则和独立正史的完整 Space。

## 不做 4：无边界访客社交

不做：

- 好友、私信、动态墙、陌生人匹配。
- 跨空间访客消息。
- 全局在线状态或公开访客社交图谱。
- 把真人扮演 NPC 扩展成访客社交入口。

允许：

- 既有单空间 NPC 认领 / 混合扮演兼容能力。
- 与空间或 NPC 相关的访客私有状态、记忆和回访关系。
- 访客主动选择的私有游玩身份和自声明性别，仅用于 NPC 互动上下文；不得用于访客匹配、公开筛选或社交图谱。

## 不做 5：传统地图 App 功能

不做：

- 路线规划、实时导航、POI 评分、商家入驻。
- 为了“地图更炫”而持续堆自绘地图资源和渲染细节。

允许：

- 用地图承载真实坐标、附近空间入口和现实氛围来源。

## 不做 6：传统游戏化系统

不做：

- 战斗系统、等级装备、竞技排行榜、阵营战、下注或匹配竞技。
- 把 `beggar` 等游玩身份扩展成带等级、装备、职业强度或公开排行的职业系统。

允许：

- 已发布的轻量文字玩法、委托、线索、选择式互动。
- Space-local 的状态、记忆、任务记录和状态卡。
- 版本化、访客私有的轻量游玩身份作为运行时 Prompt 上下文，让不同 NPC 依据自身人设和所属空间产生不同反应。

## 不做 7：不可落库的端到端 AI 幻觉

不做：

- 直接让 LLM 输出最终 UI、Schema 或正史。
- 跳过结构化中间层，产生无法缓存、无法回放、无法测试的结果。

允许：

- AI 生成候选内容、候选状态卡、预览报告或 dry-run。
- 经过确认后写入明确 Schema。

## 不做 8：历史 Space 的架空改写

不做：

- 为了增强可玩性，移动、合并、提前或延后真实事件的时间、地点、参与者、制度阶段与已知结果。
- 让玩家阻止、促成或取代已有明确结果的真实历史事件。
- 让没有同场证据的真实人物见面，或把中西主题镜像写成真实交流与因果。
- 把史料空白、AI 推测、原创对白或人物动机伪装成历史事实。
- 让跨时代知识直接改变当时的技术、制度或公开历史结果。

允许：

- 在经核验的真实时间线内，使用明确标记的原创角色、私人对话和不与史料冲突的局部事件。
- 让玩家改变原创角色的信任、关系、选择、是否得到帮助和私有回访结果。
- 将来源不足或存在实质争议的内容保留为 `待核验`，核验后再决定是否发布。

硬规则：历史 Space 当前不提供架空历史分支。任何分支只要能改变固定史实，就不得发布为历史内容。

## 快速判断

新需求方向正确，通常会强化：

- 真实坐标上的空间发现。
- 入场、NPC 引导、对话、玩法、记忆、回访。
- 角色差异、事件推进、私有记忆和回访连续性。
- 数据持久化、可导出和可测试。

新需求需要谨慎，通常表现为：

- 主要目标是做社交、导航、战斗、平台计费或地图炫技。
- 让运行时 AI 未经评审发布或改写正史。
- 让历史玩法以玩家选择或 AI 生成改写真实时间线。
- 引入无法落库、无法复核、无法回放的 AI 输出。
